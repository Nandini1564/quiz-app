# Quiz App

A simple and interactive Quiz Application built using HTML, CSS, and JavaScript. The app allows users to answer multiple-choice questions, calculate scores, and view results instantly.

---

## 🚀 Features

- Interactive Quiz Interface
- Multiple Choice Questions
- Score Calculation
- Instant Result Display
- Restart Quiz Option
- Responsive Design
- User-Friendly UI
- Dynamic Question Navigation

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
